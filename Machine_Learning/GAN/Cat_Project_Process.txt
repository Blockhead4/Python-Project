[ Main Process ] 

1. 캐릭터 데이터셋 크롤링 및 전처리
- 정형화된 데이터셋 존재 여부
- 데이터 크롤링 가능 여부
- 비정형 데이터 크롤링 및 전처리 가능여부

2. 힙스터 고양이를 활용, 랜드마크별 바운딩 박스 이미지 추출

3. 각 랜드마크별 이모티콘으로 바꾸는 로직(딥러닝 모델 구축)

4. 이미지 세그멘테이션으로 엣지 추출 후 전체적인 쉐입 추출

5. 쉐입으로 캐릭터 전체 쉐입 만드는 로직(딥러닝:Munit, VAE:오토인코딩???)


[ Alternative Process ]

1. 고양이 사진 >> 얼굴 바운딩 박스 학습(힙스터/각도에따라 랜드마크 추출하는 모델)
